常用命令

| 命令                  | 解释                                                                              |
|-----------------------|-----------------------------------------------------------------------------------|
| `make distclean`      | **清除一切编译过程产生的中间文件, 使当前目录仿佛和刚刚clone下来一样**             |
| `make [all]`          | **使用默认的平台配置文件开始编译**                                                |
| `make reconfig`       | **弹出多平台选择菜单, 用户可按数字键选择, 然后根据相应的硬件平台配置开始编译**    |
| `make config`         | **显示当前被选择的平台配置文件**                                                  |
| `make help`           | **打印帮助文本**                                                                  |
| `make <directory>`    | **单独编译被<directory>指定的目录, 或者叫编译单元**                               |
